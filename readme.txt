Displays a warning in the admin bar, if search engines are discouraged.

== Description ==

Ever forgot to disable the "Discourage Search Engine" option? _Index Check_ is here to help.



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/index-check` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
